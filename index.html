HajChat is a Twitch-Chat client, focused on easier chat moderation

Features

It offers several advantages over the standard webchat and other clients:

  • Supports all the base Twitch emotes and badges
  • Favorites (one-click access to your favorite chats)
  • Whisper integration
  • Direct access to CBennis logviewer
  • Modlogs in all the channels you are mod in
  • Chatpause on keypress
  • Customizeable banbuttons (choose amount, time and more)
  • Automated keyword highlighting in custom color
  • Choose your preferred Chatlayout

Requirements

HajChat is open source and completly free. To run the program you need Java 8 or newer. If you want to compile it yourself you can download the .zip containing the sources. Otherwise you can just download the compiled .jar here.
